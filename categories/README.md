# Kategorien fuer shortcuts.json

Die Regeln wurden in einzelne Kategorien aufgeteilt. So kannst du gezielt nur den passenden Bereich anpassen.

## Dateien

- 01_browser_controls.json
  - Browser: Ctrl+Q/W/L auf Cmd+Q/W/L
- 02_option_spotlight.json
  - Left Option tippt auf Spotlight (Cmd+Space), halten bleibt Option
- 03_altgr_symbols.json
  - AltGr-Zeichenbelegung fuer deutsches Layout (@, {, [, ], }, \\, ~, |)
- 04_a_f_ctrl_cmd_swap.json
  - Gegenseitiger Tausch von Ctrl und Cmd fuer A/F
- 05_terminal_shortcuts.json
  - Terminal-spezifische Ctrl+Shift Shortcuts (X, C, V, Undo/Redo)
- 06_system_shortcuts_except_terminal.json
  - Systemweite Ctrl-Mappings ausserhalb vom Terminal (X/C/V/Z/S/P/N/O/T/Shift+T)
- 07_navigation_and_delete.json
  - Wortnavigation und Loeschen (Ctrl+Shift+Pfeile, Ctrl+Backspace) ausserhalb vom Terminal

## Hinweis

Die Datei shortcuts.json bleibt unveraendert als bestehende Gesamtdatei erhalten.
Wenn du eine Funktion aendern willst, bearbeite die passende Datei in diesem Ordner.
